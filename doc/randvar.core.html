rand_apply(func, *args, viability=1.0, **kwargs)
Applies a function to random variable arguments, returning a random variable representing the distribution of 
return values from the function.
 
All non-random variable arguments to the function are treated as constant distributions.
 
The viability of the returned variable is the minimum of the viabilities of the arguments and the viability 
keyword argument, if given.
